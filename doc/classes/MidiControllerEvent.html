    MidiControllerEvent

    An anonymous type representing a complex MIDI controller event.

    MidiController events can be passed to anywhere that expects MIDI Data (eg: a target MIDI binding point), or can be used as a condition on a MIDI source binding point.

    controller

    Number

    Defined in MidiControllerEvent.js:58

    The associated controller number for any of the controller event kinds, or the program number for program change event kinds.

    When used as a source binding condition, this property can be set to -1 for program change events to be triggered on any program change.
